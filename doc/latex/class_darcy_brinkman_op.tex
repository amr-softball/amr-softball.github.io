\subsection{Detailed Description}
Operator for solving the Darcy-\/\+Brinkman equation. 

Operator for solving one component of \[ (alpha * aCoef(x) * I - beta * (cCoef(x) + Lap) ) phi = rho \] over an \textbf{ A\+MR} hierarchy.
